TROUBLESHOOTING
---------------

ISSUE: "Table doesn't exist" error
FIX: Run database_update.sql in phpMyAdmin

ISSUE: No products showing
FIX: Check if products exist in database

ISSUE: Can't update stock
FIX: Verify stock_quantity column exists in products table

ISSUE: Suppliers not showing
FIX: Run database_update.sql to create suppliers table

ISSUE: Orders not showing
FIX: Run database_update.sql to create orders table

QUICK TEST COMMANDS
-------------------
