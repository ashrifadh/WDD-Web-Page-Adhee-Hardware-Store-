========================================
ADHEE HARDWARE STORE - SETUP INSTRUCTIONS
========================================

STEP 1: DATABASE SETUP
-----------------------
1. Start XAMPP and ensure Apache and MySQL are running
2. Open phpMyAdmin (http://localhost/phpmyadmin)
3. Import the existing database:
   - Click "Import" tab
   - Choose file: adhee_hardware (2).sql
   - Click "Go"

4. Run the database update to add new tables:
   - Click on "adhee_hardware" database
   - Click "SQL" tab
   - Open database_update.sql file and copy all content
   - Paste into SQL query box
   - Click "Go"

STEP 2: ACCESS THE APPLICATION
-------------------------------
The PHP server is already running on port 8000.

Main Pages:
- Home Page: http://localhost:8000/Index.php
- Admin Dashboard: http://localhost:8000/admin_dashboard.php
- Customer Orders: http://localhost:8000/customer_orders.php (after login)
- Checkout: http://localhost:8000/checkout.php (after adding items to cart)

STEP 3: FEATURES OVERVIEW
--------------------------

ADMIN FEATURES (admin_dashboard.php):
✓ Dashboard Overview with statistics
✓ Product Management (Add, Edit, Delete, Update Stock)
✓ Supplier Management (Add, Edit, Delete)
✓ Assign Suppliers to Products
✓ Customer Management (View all customers)
✓ Order Tracking (View all orders, Update status)
✓ Low Stock Alerts

CUSTOMER FEATURES:
✓ Browse Products
✓ Search Products
✓ Add to Cart
✓ Checkout
✓ View Order History (customer_orders.php)
✓ View Order Details (order_details.php)

STEP 4: TESTING
---------------
1. Register a new customer account
2. Browse products and add to cart
3. Go to checkout and place an order
4. View your order history
5. Access admin dashboard to manage everything

STEP 5: ADMIN ACCESS
--------------------
To create an admin user, run this SQL in phpMyAdmin:

UPDATE users SET role = 'admin' WHERE id = 1;

(Replace id = 1 with your user ID)

NOTES:
------
- All API endpoints are in the /api folder
- Stock is automatically reduced when orders are placed
- Low stock alerts appear when stock < min_stock_level (default: 10)
- Orders have 4 statuses: pending, processing, completed, cancelled

TROUBLESHOOTING:
----------------
If you see "Table doesn't exist" errors:
- Make sure you ran database_update.sql
- Check that all tables were created in phpMyAdmin

If products don't show stock:
- Stock columns were added, existing products will have 0 stock
- Update stock quantities from admin dashboard
